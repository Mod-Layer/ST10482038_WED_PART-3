# Farmers Market Website

Part 3: JavaScript, SEO, Forms and Feedback corrections

## Part 3 features
- Lightbox image gallery (click thumbnails to open larger image)
- Google Maps embed placeholder on contact page
- Simple service/product filter dropdown on services page
- Basic enquiry form processing (client-side) with estimate display
- Formspree placeholder for contact form (add your Formspree action URL to the form `action` attribute)
- robots.txt and sitemap.xml added to root

## How to preview locally
1. Open the `IMAGES1/index.html` file in a browser (double-click or use Live Server extension).
2. Test the gallery on `services.html` and the map on `contact.html`.
